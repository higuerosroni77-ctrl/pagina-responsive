# Página Responsive para el Liceo Tecpán

Este proyecto consiste en el desarrollo de un sitio web responsive para el Liceo Tecpán, diseñado con el objetivo de proporcionar una plataforma informativa, accesible y visualmente estructurada para estudiantes, padres de familia y visitantes en general.

El sitio está construido utilizando tecnologías web modernas que permiten una correcta visualización en diferentes dispositivos, como computadoras, tablets y teléfonos móviles.

---

## Objetivos del Proyecto

- Presentar información institucional de forma clara y organizada.
- Facilitar el acceso a secciones relevantes como servicios educativos, galería, contacto y más.
- Ofrecer una experiencia de usuario amigable y adaptada a diferentes resoluciones de pantalla.
- Implementar un diseño atractivo y funcional que represente adecuadamente al Liceo Tecpán.

---

##  Tecnologías Utilizadas

- **HTML5** – estructura y contenido del sitio.
- **CSS3** – estilos visuales y diseño responsivo.
- **Bootstrap 5** – apoyo en la maquetación y componentes responsive.
- **JavaScript** – interacción básica dentro del sitio.
- **GitHub Pages** – despliegue y publicación del sitio web.

---

##  Estructura Principal del Proyecto

- `/CSS` – hojas de estilo y configuraciones visuales.
- `/js` – scripts utilizados para interacción.
- `/imágenes` – recursos gráficos del sitio.
- Archivos HTML correspondientes a cada sección principal del sitio.
